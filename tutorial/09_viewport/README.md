[Lazy Foo](https://lazyfoo.net/tutorials/SDL/09_the_viewport/index.php)

 - init
   - Create the window
   - Create the renderer
   - Set the render draw colour
   - init sdl_image
 - Load the media
 - Loop until a quit command is received
    - Set the rendering colour (clear to colour)
    - Clear the renderer
    - Set the viewports
    - Render the viewports
    - Update the screen
 - Cleanup and quit
