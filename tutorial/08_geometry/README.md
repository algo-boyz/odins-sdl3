[Lazy Foo](https://lazyfoo.net/tutorials/SDL/08_geometry_rendering/index.php)

 - init
   - Create the window
   - Create the renderer
 - Loop until a quit command is received
    - Set the rendering colour (clear to colour)
    - Clear the renderer
    - Set the renderer per primitive
    - Draw the primitives
    - Update the screen
 - Cleanup and quit
